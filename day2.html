  • To create a table use "table"
    • Featutes of a Table are:
      1. Table Header - thead
      2. Table Body - tbody
      3. Table Footer - tfoot
      4. Table Row (always in bold) - tr
      5. Table Header Cell - th
      6. Table Cell - td
